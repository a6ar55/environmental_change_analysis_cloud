About Deforestation Detection

This application uses advanced AI-powered semantic segmentation to identify forest and deforested areas in satellite imagery.

How It Works

  1. Upload: Users upload satellite or aerial images of forest regions.
  2. Cloud Processing: Images are processed in Google Cloud using Vertex AI for semantic segmentation.
  3. Analysis: Each pixel is classified as either forest (green) or deforested (red).
  4. Visualization: Results are displayed as color-coded masks and overlays.
  5. Statistics: The system calculates the percentage of forest coverage and deforested area.

Google Cloud Requirements

This application requires a Google Cloud service account with the following permissions:

  • Storage Admin - For creating and managing GCS buckets
  • Storage Object Admin - For uploading/downloading files to/from GCS
  • Vertex AI User - For using Vertex AI models
  • Vertex AI Admin - For deploying models to endpoints
If you encounter permission errors, please ensure your service account has all the required roles assigned.

Technology Stack

  • Frontend: HTML, CSS, JavaScript, Bootstrap
  • Backend: Python, Flask
  • Image Processing: OpenCV, NumPy
  • Cloud Services: Google Cloud Storage, Vertex AI
  • Deployment: Docker
